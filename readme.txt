Minimalny, nowoczesny szkielet wtyczki afiliacyjnej dla WooCommerce.

== Instalacja ==
1. Wgraj ZIP przez Kokpit → Wtyczki → Dodaj nową → Wyślij wtyczkę.
2. Aktywuj.
3. Przejdź do AffiLite → Ustawienia, ustaw stawkę prowizji itd.
4. Utwórz stronę i wstaw shortcode [aff_portal].
5. Dodatkowa opcja Bonusy
